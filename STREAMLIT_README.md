# Streamlit Frontend for Vector Autoregression

Streamlit-based frontend for Book 2, Chapter 20: Vector Autoregression prediction model.

## Features

- **Manual Prediction Form**: Input SRL variables and get one-step VAR(1) forecasts
- **Results Display**: View predictions, RMSE per variable, and input values
- **Backend Integration**: Communicates with FastAPI backend on port 9620
- **Real-time Status**: Check backend connection status
- **Responsive Design**: Works on desktop and tablet devices

## Installation

Install required packages:

```bash
pip install -r requirements-streamlit.txt
```

## Running the App

### Option 1: PowerShell
```powershell
./run_streamlit.ps1
```

### Option 2: Command Prompt
```cmd
run_streamlit.bat
```

### Option 3: Direct Streamlit
```bash
streamlit run streamlit_app.py --server.port 8621
```

## Prerequisites

- **FastAPI Backend** must be running on port 9620
  - Start it with: `python -m uvicorn backend.api:app --host 127.0.0.1 --port 9620`

- **Dashboard Data** must exist at: `outputs/backend/dashboard.json`
  - Generated by: `python backend/run_var_pipeline.py`

## Usage

1. Generate the model and dashboard:
   ```bash
   python backend/run_var_pipeline.py
   ```

2. Start the FastAPI backend:
   ```bash
   python -m uvicorn backend.api:app --host 127.0.0.1 --port 9620
   ```

3. Run the Streamlit app:
   ```bash
   streamlit run streamlit_app.py --server.port 8621
   ```

4. Open browser to **http://localhost:8621**

5. Enter values for the 5 SRL variables
6. Click "Run Prediction" to get one-step forecast
7. View results in tables

## Input Variables

The VAR(1) model operates on 5 Self-Regulated Learning (SRL) constructs:

- **efficacy**: Learner's belief in their ability
- **value**: Perceived value of the task
- **planning**: Goal setting and planning
- **monitoring**: Self-monitoring during task
- **effort**: Amount of effort invested

Data is from focal learner "Alice" over 156 days.

## Output

### One-Step Forecast
Predicted values for the next time step based on current observation.

### RMSE per Variable
Root Mean Square Error for each variable's prediction uncertainty.

### Current State (Echo)
Echo of input values submitted for verification.

## Architecture

- **Frontend**: Streamlit web application
- **Backend**: FastAPI server
- **Model**: VAR(1) via statsmodels
- **Data**: Self-Regulated Learning (SRL) from srl.csv

## Dependencies

- streamlit: Web framework
- requests: HTTP client for API calls
- pandas: Data manipulation and display
- plotly: Interactive visualizations
- numpy: Numerical computing

## Troubleshooting

**Issue**: "Dashboard data not found"
- **Solution**: Run `python backend/run_var_pipeline.py` first

**Issue**: "Backend unreachable"
- **Solution**: Start backend with `python -m uvicorn backend.api:app --host 127.0.0.1 --port 9620`

**Issue**: Port 8621 already in use
- **Solution**: Use different port: `streamlit run streamlit_app.py --server.port 8622`
